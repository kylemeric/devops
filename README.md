## Installation

1. Cloner le dépôt :
    git clone <url-du-dépôt>
2. Se placer dans le dossier :
    cd nom-du-projet
3. Installer les dépendances :
    - Exemple Node.js : npm install
    - Exemple Python : pip install -r requirements.txt
